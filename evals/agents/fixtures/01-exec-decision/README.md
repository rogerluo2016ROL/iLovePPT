# Fixture 01 · Executive Decision Deck

## 测什么

测系统在 **executive decision** 场景下的全流水线表现:
- audience = executive(语气重结论 / 数字)
- 时长 15 分钟(中等)
- 模式 speaker(现场演讲)
- 含 1 个数据 CSV + 1 个参考架构图

## 期望出来的 deck 形态

- **顶端论点**:完整推荐句(动+宾+边界)
- **章节数**:3-5,MECE
- **action title**:全是结论句,串读能讲完整故事
- **数字密度**:每页 ≥ 1 个数字
- **Source 引文**:数据 page 必标
- **audience overall_score 目标**:≥ 8.5(executive 视角苛刻)
