# Fixture 02 · Technical Architecture Deck

## 测什么

测系统在 **technical 架构讲解** 场景下:
- audience = technical(架构师 / 工程师,要看数据 + 架构图)
- 时长 30 分钟(中长)
- 模式 speaker
- 含 1 个 perf benchmark CSV + 多张架构图(matplotlib 出)

## 期望 deck 形态

- **架构图 ≥ 3 张**(总览 + 数据流 + 部署)
- **数据密集**:每页 2-3 个数字 OK(技术受众细节为王)
- **action title 含具体数据**(P99 latency / QPS / 错误率等)
- **audience visual_appeal 不重要**(数据为王);`info_density` 9+ 才合格
- **audience overall_score 目标**:≥ 8.0
