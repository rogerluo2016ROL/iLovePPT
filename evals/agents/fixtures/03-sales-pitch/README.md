# Fixture 03 · Sales Pitch Deck

## 测什么

测系统在 **sales 提案** 场景下:
- audience = sales(BD / 客户成功,关心卖点 + 对标 + CTA)
- 时长 20 分钟
- 模式 speaker
- 含竞品对比数据 + 客户 logo wall(brand assets)

## 期望 deck 形态

- **compare_pk / matrix_2x2 用得多**(对标 + 二选一冲击)

- **客户证言 / logo wall** 至少 1 张
- **audience visual_appeal ≥ 9**(sales deck 品牌感优先)
- **overall_score 目标**:≥ 8.5
