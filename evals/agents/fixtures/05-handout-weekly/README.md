# Fixture 05 · Handout 周报模式

## 测什么

测系统在 **handout 阅读手册** 模式下:
- audience = executive
- 时长 10 分钟(短 deck)
- 模式 **handout**(关键测试:无讲者,文字 5-8× 信息密度)
- 含周度数据 CSV

## 期望 deck 形态

- **每个 bullet 是完整可读句子**(handout 不写关键词)
- 字数遵守 handout 字数表(cards body ≤ 150,bullet ≤ 70 等,比 speaker mode 大 5-8 倍)
- Source 引文齐全(读者自读必须信)
- **overall_score 目标**:≥ 8.0(handout 模式 audience 评分标准跟 speaker 不同)
