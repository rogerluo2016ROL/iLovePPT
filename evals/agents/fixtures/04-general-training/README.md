# Fixture 04 · General Training Deck

## 测什么

测系统在 **general 培训** 场景:
- audience = general(非该领域专家,不懂行话)
- 时长 45 分钟(长 deck)
- 模式 speaker
- 含 `bypass_pyramid: tutorial`(教学 deck 可豁免金字塔严格性)

## 期望 deck 形态

- 章节按"知识点结构序"(不是金字塔论证)
- 类比 + 例子多
- 行话首次出现给全称
- audience 重 comprehension_5s(读者要听懂)
- **overall_score 目标**:≥ 8.0
