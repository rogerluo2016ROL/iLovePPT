# iLovePPT Eval 评分标准

对每个 deck 的每一页,按下列检查项打分（pass / fail）。

## 视觉（取自 visual-qa.md 12 项）
1. 无元素重叠
2. 无文字溢出 / 截断
3. 中文字体未 fallback（不是 Arial / 花体）
4. 标题与内容区间距合理
5. 颜色对比度足够（不深底深字 / 浅底浅字）
6. layout 与意图相符
7. 数字 / 图无 margin 偏移
8. 配色在 BRAND_* / GRAY_* 套色板内
9. 留白边界达标（离页边 ≥ 0.5"）
10. 表格无意外 banding
11. 无 emoji 误用 / 方块
12. 装饰大字号未异常换行

## 留白与对齐
13. 内容区无大片异常空白（短内容应居中 / 均布,不顶端挤）
14. 列 / 行对齐网格

## 结构
15. 页序节奏合理（封面 → 目录 → 章节 → 内容 → 总结 → 封底）
16. 图密度合理（若 plan 含 pic_text）

## 打分
每页记录 fail 的项号。一个 deck 的得分 = 总检查数 − 总 fail 数。
回归判定：新 scorecard 的 fail 项数对比 baseline,变多即回归。
